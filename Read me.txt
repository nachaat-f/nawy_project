# Apartment Listing API

This project is a backend application built with Node.js and MongoDB that provides an API for managing apartments.

## Overview
This project allows users to retrieve a list of apartments, view details of specific apartments, and add new apartments to the database.

## Installation
1. Clone the repository: `git clone <repository-url>`
2. Navigate to the project directory: `cd <project-directory>`
3. Install the required dependencies: `npm install`

## API Endpoints

- `GET /api/apartments/:id`
- `POST /api/apartments`
